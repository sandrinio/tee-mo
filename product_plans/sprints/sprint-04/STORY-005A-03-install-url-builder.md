# STORY-005A-03: `GET /api/slack/install` — Install URL Builder with Signed State

**Complexity: L2** — Standard, 4 files, known patterns (PyJWT signing reuse, FastAPI redirect response, dependency-injected auth).

---

## 1. The Spec (The Contract)

### 1.1 User Story
> As a **logged-in Tee-Mo user** on `/app`,
> I want to **click "Install Slack" and be redirected to Slack's OAuth consent page**,
> So that **I can authorize Tee-Mo to install into one of my Slack workspaces**.

### 1.2 Detailed Requirements

- **Req 1 — Endpoint shape:** `GET /api/slack/install` (FastAPI route in new file `backend/app/api/routes/slack_oauth.py`).
- **Req 2 — Authentication required:** The route MUST require a valid auth cookie. Use the existing dependency `get_current_user_id` from `backend/app/api/deps.py` (or whatever the auth-dep symbol is — verify in code first; pattern from `auth.py:/me`). Anonymous → 401.
- **Req 3 — State token:** Generate a JWT-signed state parameter with payload `{"user_id": <uuid str>, "exp": <now + 300s>}`, signed with the existing `JWT_SECRET` via `backend/app/core/security.py` helpers (reuse `create_access_token` pattern OR add a new `create_slack_state_token(user_id) -> str` next to it — prefer the latter for separation of concerns). Algorithm: HS256 (matches existing JWT setup).
- **Req 4 — Authorize URL construction:** Build `https://slack.com/oauth/v2/authorize?` with these query params (urlencoded):
  - `client_id` = `settings.slack_client_id`
  - `scope` = `app_mentions:read,channels:history,channels:read,chat:write,groups:history,groups:read,im:history` (comma-separated, exact 7 scopes from ADR-021/025)
  - `redirect_uri` = `settings.slack_redirect_url`
  - `state` = the signed token from Req 3
- **Req 5 — Redirect response:** Return `RedirectResponse(authorize_url, status_code=307)` (FastAPI). 307 (not 302) preserves the GET method on the final hop.
- **Req 6 — Router registration:** Register `slack_oauth_router` in `backend/app/main.py` alongside the existing `auth_router` and `slack_events_router`.
- **Req 7 — Pydantic state model:** In new file `backend/app/models/slack.py`, define:
  ```python
  class SlackInstallState(BaseModel):
      user_id: str
      exp: int
  ```
  (Used for typed `verify_slack_state_token() -> SlackInstallState` in STORY-005A-04. Just creating the model in this story.)
- **Req 8 — No body, no logging of state:** The route returns a redirect; do NOT log the state token (it contains a signed user_id, treat as a secret).

### 1.3 Out of Scope
- The OAuth callback (`GET /api/slack/oauth/callback`) — STORY-005A-04.
- Verifying the state on the callback side — STORY-005A-04 (the verify helper lives in the same module but the consumer is the callback story).
- Handling the case where user is already an installer of the same team — STORY-005A-04 handles re-install semantics.

### 3.3 Technical Logic

**`security.py` additions:**
```python
def create_slack_state_token(user_id: str, *, now: int | None = None) -> str:
    iat = now or int(time.time())
    payload = {"user_id": user_id, "exp": iat + 300, "iat": iat, "aud": "slack-install"}
    return _JWT.encode(payload, settings.jwt_secret, algorithm=settings.jwt_algorithm)

def verify_slack_state_token(token: str) -> SlackInstallState:
    payload = _JWT.decode(
        token, settings.jwt_secret, algorithms=[settings.jwt_algorithm], audience="slack-install"
    )
    return SlackInstallState(user_id=payload["user_id"], exp=payload["exp"])
```
**Note:** Use the module-local `_JWT` instance from existing `security.py` (the BUG-20260411 fix that uses a per-module `jwt.PyJWT()` rather than the global). Per FLASHCARDS.md.

**`slack_oauth.py` skeleton:**
```python
from fastapi import APIRouter, Depends
from fastapi.responses import RedirectResponse
from urllib.parse import urlencode
from app.api.deps import get_current_user_id
from app.core.config import get_settings
from app.core.security import create_slack_state_token

router = APIRouter(prefix="/api/slack", tags=["slack"])

SLACK_SCOPES = (
    "app_mentions:read,channels:history,channels:read,"
    "chat:write,groups:history,groups:read,im:history"
)

@router.get("/install")
async def slack_install(user_id: str = Depends(get_current_user_id)):
    s = get_settings()
    state = create_slack_state_token(user_id)
    qs = urlencode({
        "client_id": s.slack_client_id,
        "scope": SLACK_SCOPES,
        "redirect_uri": s.slack_redirect_url,
        "state": state,
    })
    return RedirectResponse(f"https://slack.com/oauth/v2/authorize?{qs}", status_code=307)
```

**`main.py` change:**

```python
from app.api.routes.slack_oauth import router as slack_oauth_router

```
